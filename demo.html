Grabatar.js Examples

Minimal

Grabatar.src('alex@cratercreative.com')

Set size

Grabatar.src('alex@cratercreative.com',125)

Use HTTPS

Grabatar.setup({ secure: true });
Grabatar.src('alex@cratercreative.com')

Set default / fallback parameter

Set's the d parameter as specified by Gravatar

Grabatar.setup({ d:
	'monsterid'
});
Grabatar.src('doesnt@exist.com')
